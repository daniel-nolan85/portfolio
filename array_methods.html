JavaScript Array Methods

A brief overview of methods that can be applied to a JavaScript array



Array.prototype.at()

Returns the item at the index of a given integer value:

const fruits = ['banana', 'apple', 'mango', 'kiwi', 'pear']
const index = 2

console.log(`The item of index number ${index} is ${fruits.at(index)}`)
// The item of index number 2 is mango

Negative integers are also supported:

const fruits = ['banana', 'apple', 'mango', 'kiwi', 'pear']
const index = -2

console.log (`The item of index number ${index} is ${fruits.at(index)}`)
// The item of index number -2 is kiwi


Array.prototype.concat()

Used to merge multiple arrays and return a new one:

const cities = ['Los Angeles', 'Arizona', 'Las Vegas', 'Miami', 'New Orleans']
const states = ['California', 'Texas', 'Nevada', 'Florida', 'Louisiana']
const citiesAndStates = cities.concat(states)

console.log(citiesAndStates)
// ['Los Angeles', 'Dallas', 'Las Vegas', 'Miami', 'New Orleans', 'California', 'Texas', 'Nevada', 'Florida', 'Louisiana']


Array.prototype.copyWithin()

Makes a copy of part of an array elsewhere within the same array and returns without modifying the length:

const nums = [1, 2, 3, 4, 5]

console.log(nums.copyWithin(2, 3)) // copies to index [2] all items from index [3] to the end
// [1, 2, 4, 5, 5]

const nums = [1, 2, 3, 4, 5]

console.log(nums.copyWithin(2, 3, 4)) // optional third argument specifies index position to stop copying from
// [1, 2, 5, 5, 5]


Array.prototype.entries()

Returns an iterator object containing key/value pairs for each index:

const colors = ['red', 'blue', 'yellow', 'cyan', 'purple']
const iterator = colors.entries()

console.log(iterator.next().value) // combines with in-built next() method
// [0, 'red']

console.log(iterator.next().value)
// [1, 'blue']


Array.prototype.every()

Returns a Boolean value based on whether all elements pass a provided test:

const nums = [18, 46, 77, 943, 14]

console.log(nums.every(val => val > 0))
// true


Array.prototype.fill()

Changes specified elements in an array to a static value:

const nums = [1, 2, 3, 4, 5]

console.log(nums.fill('x', 1, 3)) // fill with an x items from index [1] to, but not including, index [3]
// [1, 'x', 'x', 4, 5]


Array.prototype.filter()

Creates a new array with the elements that pass an implemented test:

const words = ['screwdriver', 'bus', 'cabbage', 'tinsel', 'octopus']
const shortWords = words.filter(word => word.length <= 6)

console.log(shortWords)
// ['bus', 'tinsel']


Array.prototype.find()

Returns value of the first element that passes an implemented test:

const nums = [20, 48, 6, 130, 2]

console.log(nums.find(val => val < 10))
// 6


Array.prototype.findIndex()

Returns index of the first element that passes an implemented test:

const nums = [28, 39, 71, 800, 94]

console.log(nums.findIndex(val => val > 50))
// 2


Array.prototype.flat()

Returns a new array with sub-array elements concatenated into it by a specified depth:

const nums = [1, 2, [3, 4, 5]]

console.log(nums.flat())
// [1, 2, 3, 4, 5]

const nums = [1, [2, [3, [4, 5]]]]

console.log(nums.flat(2))
// [1, 2, 3, [4, 5]]


Array.prototype.flatMap()

Returns a new array by applying a callback function to each item and then flattening the result. More efficient than calling map() and then flat() separately but returns an identical outcome:

const nums = [2, 4, 6, 8, 10]

console.log(nums.flatMap(val => val/2))
// [1, 2, 3, 4, 5]


Array.prototype.forEach()

Executes a given function once on each element of the array:

const letters = ['a', 'b', 'c', 'd', 'e']

letters.forEach(letter => console.log(letter))
// a
// b
// c
// d
// e


Array.prototype.includes()

Determines whether an array includes a particular value and returns a Boolean:

const days = ['Monday', 'Tuesday', 'Wednesday', 'Thursday', 'Friday']

console.log(days.includes('Tuesday'))
// true

console.log(days.includes('Saturday'))
// false


Array.prototype.indexOf()

Returns the first index at which an element can be found. If not found, returns -1:

const sports = ['football', 'hockey', 'tennis', 'basketball', 'boxing']

console.log(sports.indexOf('football'))
// 0

console.log(sports.indexOf('darts'))
// -1


Array.prototype.join()

Returns a string by concatenating all elements of an array, separated by a comma or specified character:

const people = ['Billy', 'Jane', 'Sarah', 'Thomas', 'Greg']

console.log(people.join())
// Billy,Jane,Sarah,Thomas,Greg

console.log(people.join(' '))
// Billy Jane Sarah Thomas Greg

console.log(people.join('-'))
// Billy-Jane-Sarah-Thomas-Greg


Array.prototype.keys()

Returns an iterator object containing keys for each index:

const months = ['January', 'February', 'March', 'April', 'May']
const iterator = months.keys()

for (const key of iterator) {
  console.log(key);
}
// 0
// 1
// 2
// 3
// 4


Array.prototype.lastIndexOf()

Returns the last index at which an element can be found. If not found, returns -1:

const pets = ['dog', 'cat', 'rabbit', 'dog', 'fish']

console.log(pets.lastIndexOf('dog'))
// 3

console.log(pets.lastIndexOf('snake'))
// -1


Array.prototype.map()

Performs a given function on each element and returns a new array with the results:

const nums = [5, 10, 15, 20, 25]

console.log(nums.map(val => val * 2))
// 10, 20, 30, 40, 50


Array.prototype.pop()

Removes last element from an array and returns it, modifying the length of the original array:

const animals = ['bear', 'jaguar', 'penguin', 'donkey', 'whale']

console.log(animals.pop())
// 'whale'

console.log(animals)
// ['bear', 'jaguar', 'penguin', 'donkey']


Array.prototype.push()

Adds one or more elements to the end of an array and returns its new length:

const vehicles = ['car', 'plane', 'scooter', 'hovercraft', 'truck']

console.log(vehicles.push('helicopter'))
// 6

console.log(vehicles)
// ['car', 'plane', 'scooter', 'hovercraft', 'truck', 'helicopter']


Array.prototype.reduce()

Executes a function on each element in order, passing the return value on to the next item, and giving a single value as a final result:

const nums = [1, 2, 3, 4, 5]

console.log(nums.reduce((prevVal, currVal) => prevVal + currVal))
// 15


Array.prototype.reduceRight()

Executes a function on each element from right to left, passing the return value on to the next item, and giving a single value as a final result:

const letters = ['a', 'b', 'c', 'd', 'e']

console.log(letters.reduceRight((prevVal, currVal) => prevVal + currVal, 'f'))
// fedcba


Array.prototype.reverse()

Reverses an array:

const nums = [1, 2, 3, 4, 5]

console.log(nums.reverse())
// [5, 4, 3, 2, 1]


Array.prototype.shift()

Removes the first element from an array and returns it, modifying the length of the original array:

const nations = ['UK', 'France', 'Australia', 'Japan', 'Cameroon']

console.log(nations.shift())
// 'UK'

console.log(nations)
// ['France', 'Australia', 'Japan', 'Cameroon']


Array.prototype.slice()

Returns a copy of a selected part of an array, leaving the original array unmodified:

const landmarks = ['Eiffel Tower', 'Great Pyramid', 'Stonehenge', 'Machu Picchu', 'Ayers Rock']

console.log(landmarks.slice(1, 3) // parameters indicate start and end indices to slice from and to
// ['Great Pyramid', 'Stonehenge']

console.log(landmarks)
// ['Eiffel Tower', 'Great Pyramid', 'Stonehenge', 'Machu Picchu', 'Ayers Rock']


Array.prototype.some()

Returns a Boolean value depending on whether at least one element in the array passes an implemented test:

const nums = [1, 2, 3, 4, 5]

console.log(nums.some(val => val % 2 === 0))
// true


Array.prototype.sort()

Sorts the elements of an array and returns a new array:

const toys = ['ball', 'yoyo', 'robot', 'train', 'hula-hoop']

console.log(toys.sort())
// ['ball', 'hula-hoop', 'robot', 'train', 'yoyo']


Array.prototype.splice()

Modifies an array by removing/replacing elements from it and/or adding new elements in their place:

const phonetics = ['alpha', 'bravo', 'charlie', 'echo', 'foxtrot']
phonetics.splice(3, 0, 'delta')

console.log(phonetics)
// ['alpha', 'bravo', 'charlie', 'delta', 'echo', 'foxtrot']


Array.prototype.toLocaleString()

Converts elements into strings using their toLocaleString methods according to language and configuration properties:

console.log(new Date().toLocaleString('en'), { timeZone: 'UTC' })
// 1/31/2022, 2:05:09 PM


Array.prototype.toString()

Converts array elements into a string:

const arr = ['a', 1, 'b', 2, 'c', 3]

console.log(arr.toString())
// a,1,b,2,c,3


Array.prototype.unshift()

Adds one or more elements to the beginning of an array and returns its new length:

const utensils = ['fork', 'whisk', 'spatula', 'tongs', 'masher']

console.log(utensils.unshift('can opener'))
// 6

console.log(utensils)
// ['can opener', 'fork', 'whisk', 'spatula', 'tongs', 'masher']


Array.prototype.values()

Returns an iterator object containing values for each index:

const months = ['January', 'February', 'March', 'April', 'May']
const iterator = months.values()

for (const value of iterator) {
  console.log(value);
}
// January
// February
// March
// April
// May
